# JavaScript სავარჯიშოები

ქვემოთ მოცემულია 20 სავარჯიშო if/else, შედარების ოპერატორების, ლოგიკური ოპერატორების და ternary ოპერატორის გასავარჯიშებლად.

1. შექმენით ორი ცვლადი: age = 17 და hasPermission = true.
   if/else-ით შეამოწმეთ: თუ age >= 18 ან hasPermission true-ია, დაბეჭდეთ "შესვლა ნებადართულია", სხვა შემთხვევაში "შესვლა აკრძალულია".

2. შექმენით num1 = 25 და num2 = 30.
   დაბეჭდეთ შედარებების შედეგები:
   num1 > num2
   num1 < num2
   num1 == num2
   num1 != num2

3. prompt()-ით შეაყვანინეთ მომხმარებელს რიცხვი.
   if/else-ით გამოიტანეთ: "დადებითია", "უარყოფითია" ან "ნულია".

4. შექმენით a = 10 და b = "10".
   დაბეჭდეთ შედეგები:
   a == b
   a === b
   a !== b

5. prompt()-ით შეაყვანინეთ ასაკი.
   თუ ასაკი ნაკლებია 13-ზე: "ბავშვი"
   თუ 13-დან 17-ის ჩათვლით: "თინეიჯერი"
   თუ 18 ან მეტი: "ზრდასრული"

6. შექმენით day = "monday" და isHoliday = false.
   if/else-if-ით შეამოწმეთ:
   თუ monday და არ არის დღესასწაული -> "სამსახურში მივდივარ"
   თუ monday და არის დღესასწაული -> "სახლში ვრჩები"
   სხვა შემთხვევაში -> "თავისუფალი რეჟიმი"

7. prompt()-ით შეაყვანინეთ ორი რიცხვი.
   if/else-ით იპოვეთ რომელი მეტია ან ტოლია.

8. prompt()-ით შეაყვანინეთ რიცხვი.
   ternary ოპერატორით დაბეჭდეთ: "ლუწია" ან "კენტია".

9. შექმენით score = 82.
   if/else-if-ით შეფასება:
   score >= 90 -> "A"
   score >= 80 -> "B"
   score >= 70 -> "C"
   score >= 60 -> "D"
   სხვა შემთხვევაში -> "F"

10. prompt()-ით შეაყვანინეთ პაროლის სიგრძე (რიცხვი).
    ternary ოპერატორით დაბეჭდეთ: თუ >= 8 "ძლიერი პაროლი", სხვა შემთხვევაში "სუსტი პაროლი".

11. შექმენით isLoggedIn = true და isAdmin = false.
    დაბეჭდეთ:
    isLoggedIn && isAdmin
    isLoggedIn || isAdmin
    !isAdmin

12. prompt()-ით შეაყვანინეთ ტემპერატურა.
    if/else-ით გამოიტანეთ:

    > 30 -> "ცხელა"
    > 20-30 -> "თბილა"
    > 10-19 -> "გრილა"
    > < 10 -> "ცივა"

13. შექმენით x = 15, y = 20, z = 20.
    if/else-if-ით შეამოწმეთ:
    - ყველაზე დიდი რიცხვი
    - არის თუ არა ორი ცვლადი ტოლი

14. prompt()-ით შეაყვანინეთ ასაკი და აქვს თუ არა ბილეთი (true/false).
    if-ით დაუშვით კინოში შესვლა მხოლოდ მაშინ, როცა ასაკი >= 16 და ბილეთი true-ია.

15. შექმენით number = 45.
    if/else-ით შეამოწმეთ:
    - იყოფა თუ არა 3-ზე
    - იყოფა თუ არა 5-ზე
    - თუ ორივეზე იყოფა დაბეჭდეთ "იყოფა 3-ზეც და 5-ზეც"

16. prompt()-ით შეაყვანინეთ თანხა.
    ფასდაკლების ლოგიკა:
    100-ზე ნაკლები -> 0%
    100-199 -> 10%
    200-299 -> 15%
    300+ -> 20%
    დაბეჭდეთ ფასდაკლების პროცენტიც და საბოლოო გადასახდელიც.

17. შექმენით username = "admin" და password = "1234".
    prompt()-ით შეაყვანინეთ ორივე.
    if/else-ით შეამოწმეთ ავტორიზაცია:
    ორივე სწორია -> "წარმატებული შესვლა"
    სხვა შემთხვევაში -> "მონაცემები არასწორია"

18. prompt()-ით შეაყვანინეთ სამი რიცხვი.
    if/else-if-ით იპოვეთ ყველაზე დიდი.
    დამატებით შეამოწმეთ: თუ სამივე ტოლია, დაბეჭდეთ "სამივე რიცხვი ტოლია".

19. შექმენით isWeekend = true, hasWork = false, weather = "sunny".
    if/else-ით დაგეგმეთ დღე:
    თუ weekend და არ აქვს სამუშაო და ამინდი sunny -> "გავალ სასეირნოდ"
    სხვა შემთხვევაში -> "სახლში დავრჩები"

20. კომბინირებული დავალება:
    prompt()-ით შეაყვანინეთ age, money, isStudent (true/false).
    მომხმარებელი მიიღებს ფასდაკლებულ ბილეთს თუ:
    age < 18 ან isStudent true-ია, და money >= 20.
    სხვა შემთხვევაში ჩვეულებრივი ბილეთია, თუ money >= 35.
    დანარჩენ შემთხვევაში -> "არასაკმარისი თანხა".
