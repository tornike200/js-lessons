# JavaScript Exercises

Here are 20 exercises to practice if/else, comparison operators, logical operators, and ternary operator.

1. Create two variables: age = 17 and hasPermission = true.
   Use if/else: if age >= 18 or hasPermission is true, print "Entry allowed", otherwise "Entry denied".

2. Create num1 = 25 and num2 = 30.
   Print the results of comparisons:
   num1 > num2
   num1 < num2
   num1 == num2
   num1 != num2

3. Use prompt() to ask the user for a number.
   Use if/else to print: "positive", "negative", or "zero".

4. Create a = 10 and b = "10".
   Print results:
   a == b
   a === b
   a !== b

5. Use prompt() to ask for age.
   If age < 13: "child"
   If 13 to 17: "teenager"
   If 18 or higher: "adult"

6. Create day = "monday" and isHoliday = false.
   Use if/else if:
   If monday and not holiday -> "I go to work"
   If monday and holiday -> "I stay home"
   Otherwise -> "Free time"

7. Use prompt() to ask for two numbers.
   Use if/else to find which is bigger or equal.

8. Use prompt() to ask for a number.
   Use ternary operator to print: "even" or "odd".

9. Create score = 82.
   Use if/else if for grades:
   score >= 90 -> "A"
   score >= 80 -> "B"
   score >= 70 -> "C"
   score >= 60 -> "D"
   otherwise -> "F"

10. Use prompt() to ask for password length (number).
    Use ternary operator: if >= 8 print "strong password", otherwise "weak password".

11. Create isLoggedIn = true and isAdmin = false.
    Print:
    isLoggedIn && isAdmin
    isLoggedIn || isAdmin
    !isAdmin

12. Use prompt() to ask for temperature.
    Use if/else to print:

    > 30 -> "hot"
    > 20-30 -> "warm"
    > 10-19 -> "cold"
    > < 10 -> "very cold"

13. Create x = 15, y = 20, z = 20.
    Use if/else if to check:
    - Find the biggest number
    - Check if two numbers are equal

14. Use prompt() to ask for age and if they have a ticket (true/false).
    Allow cinema entry only if age >= 16 and they have a ticket.

15. Create number = 45.
    Use if/else to check:
    - Is it divisible by 3?
    - Is it divisible by 5?
    - If divisible by both, print "divisible by 3 and 5"

16. Use prompt() to ask for amount.
    Discount logic:
    < 100 -> 0%
    100-199 -> 10%
    200-299 -> 15%
    300+ -> 20%
    Print discount percent and final price.

17. Create username = "admin" and password = "1234".
    Use prompt() to ask for both.
    Use if/else to check login:
    Both correct -> "Login successful"
    Otherwise -> "Wrong information"

18. Use prompt() to ask for three numbers.
    Use if/else if to find the biggest.
    Also check: if all three are equal, print "all three numbers are equal".

19. Create isWeekend = true, hasWork = false, weather = "sunny".
    Use if/else to plan your day:
    If weekend and without work and sunny weather -> "I will go for a walk"
    Otherwise -> "I will stay home"

20. Combined exercise:
    Use prompt() for age, money, and isStudent (true/false).
    Student gets discount ticket if:
    age < 18 or isStudent is true, and money >= 20.
    Otherwise: regular ticket if money >= 35.
    Otherwise: "Not enough money".
