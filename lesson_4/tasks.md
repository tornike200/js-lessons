# JavaScript Exercises - Functions

### Exercise 1

Create a function called `greet()` that prints in the console:

```
Hello!
```

Call the function.

---

### Exercise 2

Create a function `getSum(a, b)` that takes two numbers and returns their sum.

Example:

```
getSum(5, 10) // should return 15
```

---

### Exercise 3

Create a function `greetPerson(name)` that takes a name and prints:

```
Hello, [name]!
```

---

### Exercise 4

Create a function `getMultiplication(num1, num2)` that returns the product of two numbers.

---

### Exercise 5

Create a function `isEven(num)` that checks if a number is even or odd.

Print:

- If even: "even number"
- If odd: "odd number"

---

### Exercise 6

Create a function `getAge(birthYear)` that calculates age (2026 - birth year).

Example:

```
getAge(1995) // should return 31 (2026 - 1995)
```

---

### Exercise 7

Create a function `isAdult(age)` that checks if a person is an adult.

- If age >= 18: print "you are an adult"
- Otherwise: print "you are still a child"

---

### Exercise 8

Create a function `getDiscount(amount)` that calculates a discount:

- If amount < 100: 0% discount
- If amount 100-199: 10% discount
- If amount 200-299: 15% discount
- If amount >= 300: 20% discount

Print the final price after discount.

---

### Exercise 9

Create a function `factorial(n)` that calculates factorial.

Example:

```
factorial(5) // should return 120 (5! = 5 * 4 * 3 * 2 * 1)
```

Use recursion or a loop.

---

### Exercise 10

Create a function `getUserInfo(name, age, city)` that prints:

```
Name: [name]
Age: [age]
City: [city]
```

---

### Exercise 11

Create a function `getPower(base, exponent)` that raises a number to a power.

Example:

```
getPower(2, 3) // should return 8 (2^3)
```

---

### Exercise 12

Create a function `calculateTax(amount, taxRate)` that calculates tax.

Formula: amount \* (taxRate / 100)

Example:

```
calculateTax(1000, 20) // should return 200
```

---

### Exercise 13

Create a function `checkPassword(password)` that returns:

- If length >= 8 and has numbers: "strong password"
- If length >= 6: "medium password"
- Otherwise: "weak password"

---

### Exercise 14

Create a function `findMax(a, b, c)` that returns the biggest of three numbers.

### Exercise 15

Create a function `repeatString(str, count)` that repeats a string `count` times.

Example:

```
repeatString("Ha", 3) // should return "HaHaHa"
```

---

### Exercise 16

Create a function `convertToUpperCase(text)` that changes text to big letters.

Example:

```
convertToUpperCase("hello") // should return "HELLO"
```

---

### Exercise 17

Create a function `calculateAverage(num1, num2, num3)` that calculates the average of three numbers.

---

### Exercise 18

Create a function `validateEmail(email)` that checks if `@` is in the email:

- If yes: "valid email"
- If no: "wrong email"

---

### Exercise 19

Create a function `swap(a, b)` that changes the values of two numbers.

Example:

```
let x = 5;
let y = 10;
swap(x, y); // x should become 10, y should become 5
```

---

### Exercise 20

Create a function `countVowels(str)` that counts vowels in a string.

Example:

```
countVowels("hello") // should return 2 (e and o)
```
