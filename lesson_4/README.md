# Lesson 4 - Functions in JavaScript

## Description

In this lesson, you learn how to create and use functions. Functions are one of the most important tools in programming. They help structure code, avoid repeating code, and make code easier to read.

## Topics

- Create a function with `function` keyword
- Function parameters
- Call a function with arguments
- `return` statement to send back a result
- Save function result in a variable
- Function scope
- Use if/else in functions
- Recursion (recursive functions)
