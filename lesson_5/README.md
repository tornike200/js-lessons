# Lesson 5 - Arrow Functions in JavaScript

## Description

In this lesson, you learn about arrow functions. Arrow functions are a shorter and cleaner way to write functions in JavaScript. They were introduced in ES6 (2015) and are widely used in modern JavaScript.

## Topics

- Review: standard function declaration and function expression
- Arrow function syntax: `const fn = () => {}`
- Arrow function with parameters: `(a, b) => {}`
- Implicit return — no `{}` and no `return` keyword
- Arrow function with a single parameter — no `()` needed
- Arrow functions as callbacks (inside `forEach`, `map`, `filter`)
- Key difference between regular functions and arrow functions
